# Flashcard_App

React application users can create custom decks of flashcards to study topics of their choice 

Interfaces with an API to create, read, update, and delete records stored by the API

Functionality: 

Create deck: User can create a custom deck with their own title and description
Edit deck: User can edit existing deck name and description and delete the deck
Add card: user can add cards to their chosen deck
Edit card: edits existing cards of a specified deck, also able to delete chosen card

When studying a deck, "flipping" the card will display the backside of the card.
